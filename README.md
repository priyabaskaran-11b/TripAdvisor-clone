# TripAdvisor Clone 🌍

This project is a front-end clone of the TripAdvisor homepage built using HTML and CSS. The goal of this project is to recreate a real-world website layout and improve UI design skills.

## Features

* Responsive header with navigation menu
* Hero section with search bar
* Category-based sections (Hotels, Restaurants, Things to Do)
* Image cards with consistent layout and alignment
* Horizontal scrolling sections
* Banner sections for promotions
* Structured footer similar to real website

## Technologies Used

* HTML5
* CSS3 (Flexbox, layout styling)

##  Purpose

This project was created as part of a full-stack development course to practice building real-world UI layouts and improve front-end development skills.

## Key Learnings

* Layout structuring using Flexbox
* Aligning elements like real production websites
* Managing spacing and responsiveness
* Creating reusable UI components
